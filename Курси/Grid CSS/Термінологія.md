1.  **Grid Container** - це батьківський елемент, до якого застосували властивість `display: grid`. Він стає «сіткою», всередині якої його прямі дочірні елементи автоматично стають елементами сітки (grid items).
2. **Grid Item** - це прямий дочірній елемент елемента з `display: grid` (тобто Grid Container). Він автоматично розташовується всередині створеної сітки. 

```html 
<div class="container">   <!-- Це Grid Container -->
  <div>Item 1</div>       <!-- Це Grid Item -->
  <div>Item 2</div>       <!-- Це Grid Item -->
  <span>Item 3</span>     <!-- Це Grid Item -->
</div>
```
   
> Важливо: тільки **прямі** дочірні елементи стають Grid Items. Вкладені елементи всередині них — ні.

3.  **Grid Line** — це горизонтальні та вертикальні лінії, які формують структуру сітки.
4. **Grid Cell** — це одна комірка сітки між двома рядками та двома колонками.
5. **Grid Track** — це простір між двома сусідніми Grid Lines. Track може бути:  
	- рядком (row track)  
	- колонкою (column track)  
  
```css  
.container {  
display: grid;  
grid-template-columns: 200px 200px;  
grid-template-rows: 100px 100px;  
}
```
6. **Grid Area** — це прямокутна область сітки, яка складається з однієї або кількох Grid Cells. Grid Area обмежується чотирма Grid Lines: верхньою, нижньою, лівою та правою.  
  
```css  
.container {  
display: grid;  
grid-template-columns: 200px 200px;  
grid-template-rows: 100px 100px;  
}  
  
.item {  
grid-column: 1 / 3;  
grid-row: 1 / 2;  
}
```
